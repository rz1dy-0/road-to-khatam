# Sistem Road To Khatam Al-Quran V1

Version 1 fokus kepada pengurusan rekod bacaan murid.

## Cara upload ke GitHub Pages
1. Upload semua fail/folder ke root repository.
2. Pastikan struktur:
   - index.html
   - style.css
   - js/app.js dan fail JS lain
3. Settings > Pages > Deploy from branch > main > /root.
4. Buka URL GitHub Pages.

## Data
Data disimpan dalam localStorage browser. Excel digunakan untuk import/export.
